Privacy Policy

English to Korean Translator — Chrome Extension
Last updated: February 19, 2026

Summary: This extension does not collect, store, or transmit any personal data to any server owned or operated by the developer. All data stays in your browser.

1. Overview

English to Korean Translator ("the Extension") is a Chrome browser extension that extracts English words from webpages, translates them to Korean, and provides AI-powered vocabulary analysis. This privacy policy explains what data the Extension accesses, how it is used, and how it is stored.

2. Data We Do NOT Collect

  • We do not collect any personally identifiable information (name, email, address, etc.).
  • We do not track your browsing history or browsing behavior.
  • We do not use cookies or analytics trackers.
  • We do not sell, share, or transfer any user data to third parties.
  • We do not have any server or backend — there is no place for data to be sent to.

3. Data Accessed Locally

The Extension accesses the following data, all of which remains on your device:

Webpage Text Content

  • The Extension reads the visible text of the current webpage to extract English words.
  • This text is processed entirely within your browser and is not transmitted anywhere except to the third-party APIs described in Section 4 (only when you interact with the Extension).

API Keys (User-Provided)

  • The Extension requires you to provide your own API key for Claude (Anthropic) or Gemini (Google) to use the AI-powered word analysis feature.
  • Your API keys are stored locally in your browser using chrome.storage.sync.
  • API keys are never sent to any server owned by the developer. They are only sent directly from your browser to the respective API provider (Anthropic or Google) when you request word analysis.

Cached Translations & Word Details

  • Translated words and AI analysis results are cached locally in chrome.storage.local to avoid redundant API calls and improve performance.
  • This data remains entirely on your device.

Saved Words & Preferences

  • Words you bookmark for review and your settings (AI provider choice, model selection) are stored in chrome.storage.sync.
  • Chrome may sync this data across your signed-in Chrome browsers via your Google account, as part of Chrome's built-in sync feature. The developer has no access to this sync process.

4. Third-Party Services

The Extension communicates directly with the following third-party APIs from your browser:

Google Translate API (translate.googleapis.com)

  • Used for basic English-to-Korean word translation.
  • Sends individual English words to Google's public translation endpoint.
  • No API key required. No personal data is sent.
  • Subject to Google's Privacy Policy.

Anthropic Claude API (api.anthropic.com)

  • Used for detailed word and phrase analysis (grammar, idioms, examples) when Claude is selected as the AI provider.
  • Only called when you click on a word or submit a phrase — never automatically.
  • Sends the word/phrase and optional sentence context from the webpage. Uses your own API key.
  • Subject to Anthropic's Privacy Policy.

Google Gemini API (generativelanguage.googleapis.com)

  • Used as an alternative AI provider for word and phrase analysis when Gemini is selected.
  • Same usage pattern as Claude — only called on user interaction. Uses your own API key.
  • Subject to Google's Privacy Policy.

5. Permissions Used

  • activeTab — Read text from the current webpage when you open the side panel.
  • scripting — Inject a content script to extract English words from the page text.
  • sidePanel — Display the Extension's main interface as a side panel.
  • storage — Store translations, preferences, saved words, and API keys locally in your browser.
  • tabs — Detect page navigation to automatically extract words from the new page.

6. Data Retention

  • Session data (current page words, translation status) is cleared when you close the browser.
  • Cached translations and word details persist in local storage until you uninstall the Extension or clear browser data.
  • You can delete all stored data at any time by uninstalling the Extension or clearing site data in Chrome settings.

7. Children's Privacy

The Extension does not knowingly collect any data from children under 13. Since no personal data is collected from any user, this Extension is safe for use by all ages.
